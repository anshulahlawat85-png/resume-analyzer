# AI Resume Analyzer

AI Resume Analyzer is a Spring Boot based web application that analyzes resumes against job descriptions and calculates an ATS (Applicant Tracking System) score.

## Features

- Upload Resume PDF
- ATS Score Calculation
- Matched Skills Detection
- Missing Skills Identification
- Suggestions for Improvement
- Analysis History Tracking
- Dashboard Analytics
- PDF Report Download
- Dark Mode Support
- Search Analysis History
- ATS Score Charts

## Tech Stack

- Java
- Spring Boot
- Thymeleaf
- MySQL
- Bootstrap 5
- Chart.js
- Maven
